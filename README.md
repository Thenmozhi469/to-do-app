# 🚀 Modern Todo Dashboard

A modern and visually appealing productivity dashboard built with React, TypeScript, Mantine, and Framer Motion.

This project transforms a traditional todo application into a premium productivity platform featuring task organization, progress tracking, beautiful animations, and a modern user experience inspired by today's leading SaaS products.

---

## ✨ Features

### 📋 Task Management

* Create, edit, and delete tasks
* Mark tasks as completed
* Organize tasks efficiently

### 🏷️ Categories & Priorities

* Personal
* Work
* Study
* Fitness
* Shopping

Priority Levels:

* 🔴 High
* 🟡 Medium
* 🟢 Low

### 📊 Productivity Dashboard

* Task statistics
* Completion tracking
* Progress indicators
* Productivity insights

### 🎨 Modern UI/UX

* Glassmorphism design
* Colorful gradients

* Smooth animations
* Interactive task cards

### 🌙 Dark Mode

* Enhanced dark theme

### 📱 Responsive Design

* Mobile-friendly
* Tablet support

### 💾 Persistent Storage

* Local Storage support

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Thenmozhi469/to-do-app.git
```

### Navigate to the Project

```bash
cd to-do-app
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

or

```bash
npm start
```
